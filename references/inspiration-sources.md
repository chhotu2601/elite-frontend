# Design Inspiration Sources

Where to look when you need ideas. Organized by what you're looking for.

## Full Website Inspiration

### Awwwards (awwwards.com)
- **Best for:** Cutting-edge, award-winning creative sites
- **Look at:** Sites of the Day, Sites of the Month
- **Tip:** Filter by category (portfolio, corporate, e-commerce)
- **Level:** 3+ (these are pro-level sites)

### Godly (godly.website)
- **Best for:** Quick browsing — infinite scroll of beautiful sites
- **Look at:** Just keep scrolling. Save what catches your eye.
- **Tip:** Great for building a mood board quickly

### Dribbble (dribbble.com)
- **Best for:** Search by industry/style ("fintech landing page", "health app UI")
- **Look at:** Shots with high engagement
- **Tip:** Click through to see the full case study

### Pinterest (pinterest.com)
- **Best for:** SaaS landing pages, UI mockups, color palettes
- **Search:** "SaaS landing page", "dark mode dashboard", "minimal portfolio"
- **Tip:** The "similar pins" algorithm is surprisingly good

### Lapa.ninja (lapa.ninja)
- **Best for:** Landing page inspiration specifically
- **Look at:** Filter by color, category, style

### SaaS Landing Page (saaslandingpage.com)
- **Best for:** Exactly what the name says
- **Look at:** Real SaaS products, not concept designs

## Component Inspiration

### 21st.dev
- **Best for:** Individual components (buttons, cards, navs, forms)
- **Look at:** Browse by component type on the left sidebar
- **Tip:** Most components come with copy-paste code/prompts

### CodePen (codepen.io)
- **Best for:** Interactive demos, animation effects, CSS tricks
- **Search:** "glassmorphism card", "scroll animation", "hover effect"
- **Tip:** You can fork and modify any pen

### Monae
- **Best for:** Polished UI component library
- **Look at:** Pre-built component designs

### Aceternity UI (ui.aceternity.com)
- **Best for:** Animated React/Tailwind components
- **Look at:** Background effects, text animations, card designs

## Typography

### Google Fonts (fonts.google.com)
- **Best for:** Free fonts, pairing suggestions
- **Tip:** Filter by category (serif, sans-serif, display, mono)
- **Popular pairings:**
  - Inter + Source Serif Pro
  - Poppins + Merriweather
  - Space Grotesk + DM Sans
  - Playfair Display + Lato

### Fontjoy (fontjoy.com)
- **Best for:** AI-generated font pairing suggestions

### Typewolf (typewolf.com)
- **Best for:** See fonts in use on real sites

## Icons

### Heroicons (heroicons.com) — Clean, from Tailwind team
### Lucide (lucide.dev) — Fork of Feather, more icons
### Phosphor (phosphoricons.com) — Flexible, multiple weights
### Tabler Icons (tabler.io/icons) — 4,000+ free SVG icons

### Color Hunt (colorhunt.co) — Community-submitted palettes

### Use Animation (useanimations.com) — Micro-animation icons
